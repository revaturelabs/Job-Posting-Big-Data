 Team

 SOFTWARE AT CYTERA 
 Our cloud app is mission-control for our users, and it's where they spend most of their time. Here, they do everything from controlling and monitoring their live cells, to analysing data and planning experiments. This is where we can really delight users with carefully user-tested powerful features 
 We also use  on-system software  for orchestrating the underlying hardware, moving parts like pumps and imaging systems together to unlock user-facing functionality (PS It's very satisfying to see deployed code making physical things move around!) 
 THE ROLE

 Things you could expect to work on: 
 
 Building out the  React-based web app  with user-facing features such as cell workflow design, as well as backend functionality such as notifications and authentication 
 Implementing  containerized services  that run on the physical machine and coordinate between the hardware and the cloud 
 Developing  data pipelines  to upload, store and analyse data such as images in  AWS 
 Solving physical-world challenges involving the hardware, by implementing software algorithms such as autofocus 
 
 Responsibilities:
