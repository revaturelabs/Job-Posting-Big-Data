Studio Code is now availablePubliseringsdato: 16 juli, 2020A new feature in Visual Studio Code that enables you to debug Azure Stream Analytics queries using job diagramis now available.The Visual Studio Code extension for Azure Stream Analytics integrates job diagrams, metrics, diagnostic logs, and intermediate results to help you quickly isolate the source of a problem. The job diagram shows how data flows from input sources, like Event Hub or IoT Hub, through multiple query steps to output sinks. Each query step is mapped to a temporary result set defined in the script using a WITH statement.
