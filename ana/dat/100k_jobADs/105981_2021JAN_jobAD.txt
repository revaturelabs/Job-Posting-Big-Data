: 徐：慢。疾：快。不快不慢，从容自然。不文不武 : 既不能文，又不能武。不费吹灰之力 : 形容事情做起来非常容易，不花一点力气。重归于好 : 归于：回到；好：和好。指彼此重新和好。出神入化 : 神、化：指神妙的境域。极其高超的境界。形容文学艺术达到极高的成就。吹灰之力 : 吹灰尘的力气。比喻极轻微的力量。也比喻做事容易。垂手可得 : 垂：放下。放下双手就能得到。形容得来毫不费力丹青妙手 : 丹青：原指丹砂和青欔两种作颜料的矿物，后泛指颜料。多指国画大师。独一无二 : 没有相同的或没有可以相比的。鬼斧神工 : 象是鬼神制作出来的。形容艺术技巧高超，不是人力所能达到的。驾轻就熟 : 驾：赶马车。驾轻车，走熟路。比喻对某事有经验，很熟悉，做起来容易。举手之劳 : 一举手那样的辛劳。形容轻而易举，毫不费力。绝世无双 : 绝世：冠绝当代；无双：独一无二。姿才超众，天下无与伦比。炉火纯青 : 纯：纯粹。道士炼丹，认为炼到炉里发出纯青色的火焰就算成功了。后用来比喻功夫达到了纯熟完美的境界。妙手丹青 : 妙手：技能高超的人；丹青：绘画的颜料，比喻绘画的艺术。指优秀的画家。妙手回春 : 回春：使春天重返，比喻将快死的人救活。指医生医术高明。能工巧匠 : 指工艺技术高明的人。庖丁解牛 : 庖丁：厨工；解：肢解分割。比喻经过反复实践，掌握了事物的客观规律，做事得心应手，运用自如。巧夺天工 : 夺：胜过。人工的精巧胜过天然。形容技艺十分巧妙。情同手足 : 手足：比喻兄弟。交情很深，如同兄弟一样。轻车熟路 : 赶着装载很轻的车子走熟悉的路。比喻事情又熟悉又容易。手到擒来 : 擒：捉。原指作战一下子就能把敌人捉拿过来，后比喻做事有把握，不费力就做好了。手到拿来 : 指不费力气就能将必须擒拿的人捉到。手疾眼快 : 疾：迅速。动作迅速，眼光敏捷。形容机灵敏捷。手舞足蹈 : 蹈：顿足踏地。两手舞动，两只脚也跳了起来。形容高兴到了极点。也指手乱舞、脚乱跳的狂态。熟能生巧 : 熟练了，就能找到窍门。随心所欲 : 随：任凭；欲：想要。随着自己的意思，想要干什么就干什么。唾手而得 : 唾手：往手上吐唾沫。比喻极容易得到的东西。瓮中捉鳖 : 从大坛子里捉王八。比喻想要捕捉的对象已在掌握之中。形容手到擒来，轻易而有把握。小菜一碟 : 指不值得大肆宣扬。形容事情很容易办成。信手拈来 : 信手：随手；拈：用手指捏取东西。随手拿来。多指写文章时能自由纯熟的选用词语或应用典故，用不着怎么思考。心手相应 : 心和手相配合。形容手法熟练，心里怎么想，手就怎么做。心灵手巧 : 心思灵敏，手艺巧妙（多用在女子）。眼疾手快 : 形容做事机警敏捷。易如反掌 : 象翻一下手掌那样容易。比喻事情非常容易做。游刃有余 : 刀刃运转于骨节空隙中，在有回旋的余地。比喻工作熟练，有实际经验，解决问题毫不费事。运斤成风 : 运：挥动；斤：斧头。挥动斧头，风声呼呼。比喻手法纯熟，技术高超。运用自如 : 自如：活动不受阻碍。形容运用得非常熟练自然。左右逢源 : 逢：遇到；源：水源。到处遇到充足的水源。原指赏识广博，应付裕如。后也比喻做事得心应手，非常顺利。爱不释手 : 释：放下。喜爱得舍不得放手。悠悠伏枕 : 悠悠：长久，遥远；伏枕：伏卧在枕头上。长时间在枕头上辗转，不能入睡。忧谗畏讥 : 担忧被谗言中伤。忧深思远 : 深刻忧虑，长远打算。忧心如酲 : 酲：酒醉后神志不清的状态。心中忧闷，就像醉酒后的病态一样。形容心情十分愁闷压抑。悠悠荡荡 : 形容摇摇晃晃，飘浮不定。现多形容不勤奋。忧患余生 : 忧患：困苦患难；余生：大灾难后侥幸存活的生命。指饱经患难之后侥幸保全下来的生命。悠游自在 : 形容悠闲而舒适。同“悠游自得”。悠悠扬扬 : 形容声音高低起伏、持续和谐或事物慢节奏运动。有一得一 : 不加也不减，有多少是多少。悠哉游哉 : 指悠闲自在。有一利即有一弊 : 弊：弊病，害处。在这一方面有好处，在另一方面就会有坏处。有一搭没一搭 : 表示故意找话说。也表示可有可无，无足轻重。悠然自得 : 悠然：闲适的样子；自得：内心得意舒适。形容悠闲而舒适。有两下子 : 比喻有点本领。有以教我 : 提出问题，希望别人帮助解答。悠闲自在 : 闲暇舒适，无忧无虑。悠悠忽忽 : 形容悠闲懒散或神志恍惚的样子。有了存孝，不显彦章 : 存孝、彦章：李存孝、王彦章，五代时人，有勇力。有了李存孝，就显不出王彦章。比喻两雄不能并立。悠然神往 : 神往：心里向往。形容心神要飞到某个去处。悠游自得 : 悠游：闲适的样子；自得：内心得意舒适。形容悠闲而舒适。有以善处 : 指正确对待，妥善处理。有一手儿 : 有专长。男女之间不正当的关系。有事之秋 : 指战争或多事故的年头。有三有俩 : 指有富余钱财。悠然自适 : 形容悠闲自在，神情从容的样子。有仙则名 : 山中有了仙人居住，山就成为名山有一无二 : 只有一个，不会有第二个。指事物独特，极难得。有你不多，无你不少 : 指无关紧要的人，可有可无。有仇不报非君子 : 有仇不报不算大丈夫。形容人气质刚毅。有为有守 :
