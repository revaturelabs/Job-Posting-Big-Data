阿娇：指汉武帝刘彻的姑母（长公主刘娇）的女儿。原指汉武帝刘彻要用金屋接纳阿娇为妇。这里泛指美丽高贵的女子。闭月羞花 : 闭：藏。使月亮躲藏，使花儿羞惭。形容女子容貌美丽。彩云易散 : 美丽的彩云容易消散。比喻美满的姻缘被轻易拆散。姹紫嫣红 : 姹、嫣：娇艳。形容各种花朵娇艳美丽。沉鱼落雁 : 鱼见之沉入水底，雁见之降落沙洲。形容女子容貌美丽。逞娇呈美 : 指花草显示出美丽的颜色。楚楚动人 : 楚楚：鲜明整洁的样子。形容美好的样子引人怜爱。绰约多姿 : 绰约：姿态优美的样子。形容女子体态的美。大家闺秀 : 旧指世家望族中有才德的女子。也泛指有钱有势人家的女儿。婀娜多姿 : 形容姿态柔和而美好。粉妆玉琢 : 白粉装饰的，白玉雕成的。形容女子妆饰的漂亮或小孩长得白净。也用来形容雪景。风华绝代 : 风华：风采才华；绝代：冠绝当世。意思是风采才华为当世第一。风姿绰约 : 绰约：形容女子体态柔美的样子。指女子体态柔美丰满。姑射神人 : 姑射：山名；神人：得道的人。原指姑叶山的得到真人。后泛指美貌女子。国色天香 : 原形容颜色和香气不同于一般花卉的牡丹花。后也形容女子的美丽。华如桃李 : 形容容颜非常美丽。花容月貌 : 如花似月的容貌。形容女子美貌。花枝招展 : 招展：迎风摆动的样子。形容打扮得十分艳丽。及笄年华 : 笄：古代盘头发用的簪子。古代女子已订婚者十五而笄；未订婚者二十而笄。指少女到了可以出嫁的年龄。巾帼丈夫 : 巾帼：古代妇女配戴的头巾和发饰，后借指妇女。有大丈夫气慨的女子。绝代佳人 : 绝代：当代独一无二；佳人：美人。当代最美的女人。绝世独立 : 绝世：当代独一无二。当世无双，卓然而立。多用来形容不同凡俗的美貌女子。郎才女貌 : 郎：旧指女子对丈夫或情人的称呼。男的有才气，女的有美貌。形容男女双方很相配。面如傅粉 : 形容男子美貌。面如冠玉 : 比喻男子徒有其表。也用来形容男子的美貌。袅娜娉婷 : 形容女子姿态美好的样子。亦借指美人。袅袅婷婷 : 袅袅：柔美貌；婷婷：美好貌。形容女子姿态柔美。袅袅娉娉 : 袅袅：柔美貌；娉娉：美好貌。形容女子姿态柔美。娉娉袅袅 : 娉娉：秀美的样子；袅袅：细长柔美的样子。形容女子苗条俊美，体态轻盈。娉婷袅娜 : 形容女子姿态柔美。亦借指美人。娉婷婀娜 : ①形容女子姿态柔美。②借指美人。千娇百媚 : 娇、媚：美好。形容女子姿态美好。倾国倾城 : 倾：倾覆；城：国。原指因女色而亡国。后多形容妇女容貌极美。人面桃花 : 形容男女邂逅钟情，随即分离之后，男子追念旧事的情形。柔枝嫩条 : 柔嫩的枝条。比喻温柔美貌的妙龄女子。柔枝嫩叶 : 柔软娇嫩的枝叶。比喻温柔美貌的妙龄女子。如花似月 : 形容女子姿容出众。同“如花似玉”。色衰爱弛 : 色：姿色、容颜；弛：松懈，衰退。指靠美貌得宠的人，一旦姿色衰老，就会遭到遗弃。指男子喜新厌旧。桃腮杏脸 : 形容女子容貌美丽。同“杏脸桃腮”。桃羞杏让 : 桃花和杏花都感到羞愧，只好退让。形容女子比花还要艳丽动人。福衢寿车 : 通往福寿的道路和车子。指获取福寿的途径。腐肠之药 : 指肥肉、好酒。腐木不可以为柱 : 腐朽的木料不能用做支柱。比喻愚笨、能力较差的人不能担任要职或委以重任。腹中鳞甲 : 鳞甲：比喻人多巧诈之心。比喻居心险恶。肤如凝脂 : 皮肤象凝固的脂。形容皮肤洁白且细腻。簠簋之风 : 簠簋：古代食器、祭器。指官吏贪财受贿的风气。腐索驭马 : 索：粗的绳子。指以腐朽的绳索驾驭奔驰着的骏马。形容存在着被颠覆的危险。腹中甲兵 : 甲兵：穿着盔甲的士兵。腹中装有士兵。比喻人有雄才大谋略。腐索捍马 : 索：粗的绳子；捍：拉、套。腐烂的粗绳子套马。比喻小心谨慎地处理政事。腹心股肱 : 腹心：心腹，比喻左右亲信；股肱：比喻帝王身边的得力臣子。形容十分亲近且办事得力的人。腹中兵甲 : 兵甲：即甲兵，穿着盔甲的士兵。腹中装有士兵。比喻人有雄才谋略。肤泛不切 : 肤：肤浅；泛：一般；不切：不切合实际。指文章或言论没有深刻切实的内容。腹心之患 : 比喻严重的祸患。肤见謭识 : 浅陋的见识。腹心内烂 : 内心溃烂。比喻国家或集团等内部分崩离析。腹背夹攻 : 腹背：比喻前后两面。前后受到敌人的夹攻。腹心相照 : 腹心：内心；照：映照，见。以真心相见。比喻彼此很知心，达到心心相印的程度。腹热心煎 : 形容心中焦急。腹热肠荒 : 元曲俗语。形容焦急、慌乱。同“腹热肠慌”。腹背受敌 : 腹：指前面；背：指后面。前后受到敌人的夹攻。腹热肠慌 : 元曲俗语。形容焦急、慌乱。腹饱万言 : 指学识渊博。腹诽心谤 : 腹诽：口里不说，心里不满。指心怀不满，暗中发泄。腹里地面 : 指内部地区。艴然大怒 : 艴然：恼怒地。形容人十分恼怒腹载五车 : 比喻读书甚多，知识渊博。腹背相亲 : 腹背：指关系密切。比喻双方关系亲密。蝮螫解腕 :
