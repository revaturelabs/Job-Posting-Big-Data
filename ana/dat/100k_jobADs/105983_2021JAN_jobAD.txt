终日：从早到晚，一天。一天都过不下去。形容局势危急或心中极其恐慌不安。不知所措 : 错：错开；处置。不知道怎么办才好。形容处境为难或心神慌乱。擦拳磨掌 : 形容准备动武，准备动手干或焦灼不安的样子。诚惶诚恐 : 诚：实在，的确；惶：害怕；恐：畏惧。非常小心谨慎以至达到害怕不安的程度。喘月吴牛 : 比喻因受某事物之苦而畏惧其类似者。大吃一惊 : 形容对发生的事感到十分意外。大惊失色 : 非常害怕，脸色都变了。怛然失色 : 怛：畏惧，恐惧。指因害怕而变脸色。担惊受怕 : 担心害怕。指常处在惊吓、恐惧之中。担惊受恐 : 担心害怕。指常处在惊吓、恐惧之中。担惊忍怕 : 担心害怕。指常处在惊吓、恐惧之中。胆战心寒 : 战：发抖。形容害怕之极。胆寒发竖 : 形容恐怖之极。胆裂魂飞 : 裂：破裂。魂：魂灵、神志。胆也破了，魂也飞了。形容极度害怕、恐慌。吊胆提心 : 提心吊胆。 形容非常担心或害怕。儿女英雄 : 指感情丰富，年轻有作为的英雄人物。骨寒毛竖 : 形容十分害怕。骇人听闻 : 骇：震惊。使人听了非常吃惊、害怕。寒心酸鼻 : 寒心：心中战粟；酸鼻：鼻子辛酸。形容心里害怕而又悲痛。寒毛卓竖 : 汗毛都竖立起来。形容非常恐怖，或心情特别紧张害怕。汗不敢出 : 吓得连汗都不敢往外冒了。形容紧张害怕到了极点。汗流浃背 : 形容非常恐惧或非常害怕。同“汗流浃背”。汗洽股栗 : 汗流浃背，两腿颤抖。形容害怕、恐惧的样子。惶恐不安 : 惶：恐惧。内心害怕，十分不安。惶惶不安 : 惶：恐惧。内心害怕，十分不安。惶惶不可终日 : 惊慌地连一天都过不下去。形容惊恐不安到了极点。魂飞胆战 : 形容惊恐万分，极端害怕。同“魂飞魄散”。魂飞魄散 : 吓得连魂魄都离开人体飞散了。形容惊恐万分，极端害怕。噤若寒蝉 : 噤：闭口不作声。象深秋的蝉那样一声不吭。比喻因害怕有所顾虑而不敢说话。兢兢战战 : 兢兢：小心谨慎的样子；战战：畏惧的样子。形容害怕而小心谨慎的样子。惊心动魄 : 使人神魂震惊。原指文辞优美，意境深远，使人感受极深，震动极大。后常形容使人十分惊骇紧张到极点。惊心吊胆 : 形容十分害怕或担心。惊恐万状 : 惊恐：害怕；万状：各种样子，表示程度极深。形容害怕到了极点。惊愕失色 : 失色：由于受惊或害怕而面色苍白。惊恐得改变了神色。形容十分吃惊。惊慌失措 : 失措：失去常态。由于惊慌，一下子不知怎么办才好。惊惶万状 : 惊惶：害怕；万状：各种样子，表示程度极深。形容害怕到了极点。局促不安 : 局促：拘束。形容举止拘束，心中不安。恐后争先 : 指害怕落后，追求上进。恐慌万状 : 恐：害怕；慌：惊慌；状：情状。形容害怕惊慌到极点。泪如雨下 : 眼泪象雨水似的直往下流。形容悲痛或害怕之极。黄童白叟 : 黄发儿童，白发老人。泛指老人与孩子。黄杨厄闰 : 黄杨：树木名；厄：困苦；闰：闰年。旧时传说，黄杨木难长，遇到闰年，非但不长，反而会缩短。比喻境遇困难。黄耳寄书 : 黄耳：犬名；书：书信。黄耳为主人传递家信。指传递家信。黄皮寡廋 : 形容面黄肌瘦的样子。同“黄皮刮廋”。黄绢幼妇 : “绝妙”二字的隐语。黄童白颠 : 泛指老人与孩子。同“黄童白叟”。黄童皓首 : 泛指老人与孩子。见“黄童白叟”。黄衣使者 : 指出使宫市的太监。黄袍加身 : 五代后周时，赵匡胤在陈桥兵变，部下诸将给他披上黄袍，拥立为天子。后比喻发动政变获得成功。黄金世界 : 形容美好完善的境地。黄金时间 : 指广播电视在一天中收听、收视人数最多的时间。黄金时代 : 指人一生中最为宝贵的时期。也指国家政治、经济、文化最繁荣的时期。黄绵袄子 : 比喻冬天的太阳。黄钟毁弃 : 黄钟：黄铜铸的钟，我国古代音乐有十二律，阴阳各六，黄钟为阳六律的第一律；毁：毁坏；弃：抛弃。比喻贤人遭受摈斥。黄花晚节 : 黄花：菊花；晚节：晚年的节操。比喻人晚节高尚。黄花女儿 : 旧指未婚的青年女子。黄道吉日 : 迷信的人认为可以办事的吉利日子。黄金铸象 : 铸：铸造。用黄金铸造人像。表示对某人的敬仰或纪念。黄耳传书 : 黄耳：犬名；书：书信。黄犬为主人往返传书。后比喻传递家信。黄茅白苇 : 本指盐碱地里生长的茅草和芦苇。后比喻平庸之人或物。黄钟长弃 : 比喻贤才不用。同“黄锺毁弃”。黄风雾罩 : 犹言不守本分，胡乱行事。黄金失色 : 黄金失去了其明亮的色彩。旧时比喻人的命运不佳，有再大的才能也不能显示出来。黄门驸马 :
