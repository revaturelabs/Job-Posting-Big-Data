輕鬆掌握職涯發展Evernote Team • 九月 30, 2016找工作很容易讓人感到巨大壓力。不論您是初次踏入職場的新鮮人、想要轉換跑道的老鳥、或正在思考轉職，Evernote 都可以幫助您掌握求職過程。從收集職缺、整理申請文件、保存履歷、作品集、推薦信、聯絡人資訊等，讓 Evernote 為您分攤重擔，助您順利展翅高飛。建立自己專屬的職涯資料夾成功的關鍵往往就是看平日的準備有多周全。事前多花點時間，就可以在實際找工作時更快，更輕鬆，更少點痛苦。1. 建立一份職涯專屬的記事本第一步是建立一個 Evernote 記事本，收納您所有工作相關的資料。記事本標題隨您取。一開始裡面可能有點空，但先不用太緊張。點此看 Evernote 介面操作介紹（Windows） （Mac）建立好記事本後，想想找工作常需要的「材料」。申請工作時會需要哪些資料？履歷表自傳、求職信、動機說明先前工作的範本、作品集照片您的資料往往會是各種不同檔案格式：Word 檔、PDF 檔、圖檔、甚至多媒體檔。但不論格式為何，您都可以存入同一則記事中。這時就要提到下一步了。2. 建立一則工作紀錄的專門記事在《專業》記事本中建立一則新記事並取好標題。這就是您的「職涯資料夾記事」，裡面保存您所有履歷、自傳、作品範本等。專業小提示：在記事標題前加個「@」就可以將記事置頂，方便隨時繼續新增資料。您的職涯資料夾記事就是用來集中保存所有申請工作資料的地方。如果已經有之前的履歷、自傳、作品、相片、影音檔等等，全都可以用附件或連結的形式加到此記事中。3. 維護您的職涯資料夾記事假如您對現在的工作很滿意，真心恭喜您。但我們依然建議人人都應該隨時保持履歷更新。一般來說，一年至少要更新一次。舊版也別刪掉，可以偶爾回顧，看看自己在工作方面的成長。如果您的職涯路線尚未完全定型，建議您可以存有多幾種求職信範本，用在不同的職位申請上。真正申請工作時範本可能還是需要做調整，但是絕對會比沒有範本時省下許多時間。除了履歷和求職信，您還可以在職涯資料夾記事中建立一個「作品集」區塊。裡面是所有可以展示給潛在雇主看的資料。平時只要完成了一個讓您特別自豪的專案，就可以加註完成日期，把它放到此區中。在職涯資料夾記事中建立好這些基本資料後，其他需要的資料例如大頭照、數位簽名檔也都可以加進去。在 Evernote 裡隨時準備好一份職涯資料夾記事，就可以免除緊急更新履歷、建立新求職信、翻找先前的工作資料等讓人頭痛的情況。好的工作機會往往可遇不可求，所以更要在平日做好準備工作，需要時就能立即派上用場！專業小提示：除了作品集和履歷，Evernote 也是您開發、打磨自己「電梯簡報」（elevator pitch）的絕佳所在。隨時準備好幾個談話重點，讓您的面試可以更順利。利用 Web Clipper 收集資料您可能仰賴數個工作搜尋引擎或類似 LinkedIn 等的聚合器來尋找適合的工作，這時就可以使用 Evernote Web Clipper 將您看到有興趣的工作剪藏下來。此外，看到職涯相關的文章或建議等也可以用 Web Clipper 擷取收藏。使用 Web Clipper 從任何網站擷取工作都很容易。下載安裝好 Evernote Web Clipper 瀏覽器擴充套件後，看到有興趣的資料就點擊 Web Clipper（大象）按鈕就可以儲存網頁（或任何一部份）到 Evernote。點擊「+」或「-」號就能調整擷取資訊的多寡。小提示：點擊「簡化文章」選項，Web Clipper 就會為您捨棄網頁上其他的橫幅或格式，只留下簡單純粹的文章本身。確認擷取資料的範圍後，記得命名。這份擷取資料會以您所訂的標題名稱存成 Evernote 裡的一則記事。您可以建立一套命名系統，包含職缺發佈時間、公司名稱、頭銜等資訊。[職缺發佈日期 YYYY MM DD] [公司名稱] [職位]當您使用日期命名記事，並且使用標題排序記事，所有記事就會依日期排列，方便您決定申請工作的優先順序。最後在按下「確定」按鈕前，別忘了確認一下儲存到的記事本就是您先前建立的《專業》記事本。另外，您還可以直接貼上標籤或加點自己的評語或備註。遞交申請、追蹤進度除了網路上的職缺廣告，可能會有一些是透過認識的人介紹或獵人頭公司得知的。別忘了為這些工作機會分別建立記事，記下聯絡人資訊以及工作內容等。所有資料都集中一處後就要專注在實際申請上了。如果您已經依照上述方法建立職涯資料夾，已然省下許多時間。現在只要針對個別工作稍微調整一下求職信，所有其他資料都不用再煩惱。遞出職位申請後，把您所有申請資料留一份副本在 Evernote 裡。這樣一來，當對方回覆時，您可以清楚知道自己交出了哪些資料，避免搞錯公司等尷尬狀況。每當有新發展都別忘了更新您的記事，隨時記下聯絡人資訊、申請狀態、公司研究筆記、還有面試時要提問的問題。排好面試時間後，也別忘了建立提醒事項。專業小提示：如果您申請資料中有特別哪一部份獲得正面回響，記得記下來。特別成功的求職信可以移入職涯資料夾記事。有獲得面試機會的筆記記事要保留起來。此外，如果您在面試時被問到意料之外的問題，記下來，為下次做更好的準備。使用 Evernote 不僅對您找工作的過程起到助益，您也因此擁有一份自己完整的工作檔案、記事、作品集等隨時可以翻閱參考了。Share ArticleAsset 11以 Evernote 專業版為您的記事升級。升級專業版顯示更多 使用案例 的相關文章使用案例38 樣您應該存入 Evernote 的東西剛開始接觸 Evernote 嗎？還是已經用了一陣子？提供您一些智慧好方法，讓 Evernote
