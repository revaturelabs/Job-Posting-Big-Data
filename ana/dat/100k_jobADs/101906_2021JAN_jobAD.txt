形容人到中年对亲友离别的伤感情绪。敝帚自珍 : 敝：破的，坏的；珍：爱惜。把自己家里的破扫帚当成宝贝。比喻东西虽然不好，自己却很珍惜。别鹤孤鸾 : 别：离别；鸾：凤凰一类的鸟。离别的鹤，孤单的鸾。比喻远离的夫妻。不吝指教 : 不要舍不得指点教导。不吝珠玉 : 吝：舍不得。不要舍不得好的东西。恳切希望别人给予指教的谦词。不屑一顾 : 不屑：不值得，不愿意；顾：看。认为不值得一看。形容极端轻视。不忍释手 : 舍不得放手。形容十分喜爱，缠绵缱绻 : 缠绵：心绪郁结；缱绻：难舍难分。形容感情深厚，难舍难分。持之以恒 : 持：坚持；恒：恒心。长久坚持下去。迟迟吾行 : 迟迟：慢慢地。原指慢慢考虑考虑再走。形容恋恋不舍地离开。出内之吝 : 舍不得拿出去，出手小气。同“出纳之吝”。出纳之吝 : 舍不得拿出去，出手小气。大模大样 : 形容态度傲慢，目中无人的样子。睹物思人 : 睹：看；思：思念。看见死去或离别的人留下的东西就想起了这个人。读不舍手 : 舍：舍得，愿意。读书读到有兴趣时舍不得放下。分秒必争 : 一分一秒也一定要争取。形容充分利用一切时间。故土难离 : 故土：出生地，或过去住过的地方，这里指故乡、祖国。难于离开故乡的土地。形容对家乡或祖国有无限的眷恋之情。化腐朽为神奇 : 神奇：神妙奇特的东西。变坏为好，变死板为灵巧，变无用为有用。还珠买椟 : 比喻没有眼力，取舍不当。同“买椟还珠”。积财吝赏 : 吝：使不得。尽管积累了很多财富，却舍不得赏赐给下属。形容非常吝啬。俭以养廉 : 俭：节俭；廉：廉洁。节俭可以培养廉洁的作风。坚持不懈 : 懈：松懈。坚持到底，一点不忪懈。进退两难 : 前进和后退都难。比喻事情无法决定，因而难以行动。乐不思蜀 : 椟：木匣；珠：珍珠。买下木匣，退还了珍珠。比喻没有眼力，取舍不当。迷恋骸骨 : 迷恋：过度爱好而难以舍弃；骸骨：尸骨。迷恋着尸骨。比喻舍不得丢弃陈旧腐朽的事物。迷而知反 : 迷路后知道回来。比喻有了过失能够改正。迷途知返 : 迷途：迷失道路；返：回来。迷了路知道回来。比喻发觉自己犯了错误，知道改正。目断魂消 : 因望不到而内心伤悲。形容离别的悲伤愁苦。难以忘怀 :
