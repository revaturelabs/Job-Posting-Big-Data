La configuration du module de prise de rendez-vous se fait en 4 étapes :Paramètres du moduleRendez-vous dans le module ParamètresSélectionnez le module Paramètres du moduleVous pouvez alors éditer un message d'accueil aux étudiants dans chacune des langues disponibles sur votre Career CenterConfigurer un délai imposé aux étudiants entre deux rendez-vous (en jours). Cette restriction ne s'applique qu'aux demandes à l'initiative de l'étudiant.Ajout des conseillersRendez-vous dans le module ParamètresSélectionnez le module ConseillersCliquez sur \"Ajoutez un conseiller\"Sélectionnez un administrateur du Career Center puis éditez son profil en étant le plus exhaustif possible. Puis cliquez sur \"Créer le conseiller\".A noter que si vous travaillez avec des coachs externes, vous pouvez leur créer un profil administrateur \"Conseillers carrière\" afin qu'ils aient un accès administrateur limité aux modules Rendez-vous et Inscrits.Création des types de Rendez-vous proposésRendez-vous dans le module ParamètresSélectionnez le module Types de RDVCliquez sur \"Ajouter un type de rendez-vous\"Complétez le titre et sélectionnez la durée du rendez-vous proposé.Vous pouvez également décider de ne rendre visible le type de rendez-vous qu'à certains programmes (il vous suffit de le renseigner dans le champs \"Programmes ciblés\".Vous pouvez également décrire le rendez-vous proposé. Cliquez sur \"Créer le type de rendez-vous\" pour finaliser la création.Ajout des disponibilités du conseillerRendez-vous dans le module \"Rendez-vous\"Cliquez sur le calendrier des disponibilitésCliquez sur \"Créer une disponibilité\"Sélectionnez le type de rendez-vousChoisissez le campusIndiquez la salle du rendez-vousDéfinissez l’heure de début et de finDéfinissez la date et le rythme de répétition de la disponibilitéCliquez sur \"Ajouter\".La disponibilité créée sera alors visible de l'étudiant qui pourra prendre rendez-vous sur ce créneau.Dans un premier temps le module de prise de rendez-vous est activé uniquement côté administrateurs, le temps de la configuration. Une fois la configuration terminée, nous vous invitons à contacter votre interlocuteur JobTeaser afin qu'il active le module côté étudiants.
